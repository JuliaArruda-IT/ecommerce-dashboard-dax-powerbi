# ecommerce-dashboard-dax-powerbi

🛒 Dashboard de E-commerce com Power BI e Fórmulas DAX
Desafio prático da Formação Power BI Analyst na DIO — modelagem e transformação de dados com DAX, criando um modelo Star Schema a partir da tabela Financial Sample.

📋 Descrição:
Neste projeto, a tabela única Financial Sample é desmembrada em tabelas dimensão e fato, construindo um modelo dimensional em estrela no Power BI. O processo envolve criação de tabelas por agrupamento, colunas condicionais e a função DAX CALENDAR() para a dimensão de datas.
